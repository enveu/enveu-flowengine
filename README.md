## Give wings to your media workflows.

Media teams work across disconnected systems:

- Video platforms
- AI services
- Storage systems
- Metadata tools
- CMS platforms
- Social platforms
- Notification systems

Most teams still rely on scripts, spreadsheets, and manual handoffs.

Flow sits in the middle of your stack and orchestrates everything.

```text
┌────────────┐
│ Triggers   │
│ Webhooks   │
│ CMS Events │
│ Schedules  │
└─────┬──────┘
      │
      ▼

┌─────────────────────┐
│ Enveu FlowEngine    │
│                     │
│ Ingest              │
│ Process             │
│ Transform           │
│ Publish             │
└─────┬───────────────┘
      │
      ▼

┌─────────────────────┐
│ Outputs             │
│                     │

│ OTT Platforms       │
│ Metadata            │

```

# Workflow Example

### Live → VoD automation

```text
Live Event Ends
      ↓
Recording detected
      ↓
Transcoding starts
      ↓
AI metadata generation
      ↓
Subtitles generated
      ↓
CMS publishing
      ↓
Slack notification sent
```

**Result:** Under 25 minutes from stream end to published VoD.
